Drop your MP3 audio files here with these exact names:

  menu_start.mp3      - Plays when /play menu opens
  battle_start.mp3    - Plays when a game begins
  battle_end.mp3      - Plays when a game ends
  physical_attack.mp3 - Plays for non-arcane damage cards (Blast 'Em, Tommy Gun, Raid, Sacrificial Lamb)
  magical_attack.mp3  - Plays for arcane damage cards (Dark Ritual, Mind Burn, From Beyond, etc.)
  non_attack.mp3      - Plays for utility/healing/buff cards
  monster_attack.mp3  - Plays during end-of-turn monster combat
  insanity.mp3        - Plays when a player goes insane

Audio only plays if the user is in a voice channel when they start a game.
The bot will auto-join the voice channel and auto-leave 5 seconds after game ends.
